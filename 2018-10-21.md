### Algorithm

 ***题目***  
[69. Sqrt(x)](https://leetcode.com/problems/sqrtx/) 

基本思路：
我们知道必定存在这么两个整数a和b,a^2 <= sqrt(x) <= b^2，所以我们要做的其实就是缩小这个ab的范围。使用二分法解这题时，通过比较mid^2和x大小来确定我们在哪一个半片中搜索。

下面是Java的代码实现：

```
public int mySqrt(int x) {
    long start = 1, end = x;
    while (start + 1 < end) {
        long mid = start + (end - start) / 2;
        if (mid * mid <= x) {
            start = mid;
        } else {
            end = mid;
        }
    }

    if (end * end <= x) {
        return (int) end;
    }
    return (int) start;
}
```
